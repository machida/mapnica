## Mapnicaとは
Mapnicaは歴史の出来事を地図上に表示するサービスです。<br>
年代やキーワードでの検索が可能で、検索した出来事が地図上に表示されます。

## ローカルでの立ち上げ方

```
$ git clone https://github.com/masamune17/mapnica.git
```

```
$ cd mapnica
```

```
$ bin/setup
```

```
$ rails s
```

上記を実行後[localhost](http://localhost:3000/)にアクセス

## test

```
$ bin/test
```

## Lint
```
$ bin/lint
```
